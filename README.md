# Whatpl (2024.02 ~ 진행 중)

### ✋ 프로젝트 소개

- 사이드 프로젝트 구인/구직 웹 서비스입니다.
- 사용자가 모집자에게 프로젝트 참여 요청 후, 모집자가 승인할 경우 프로젝트의 일원이 됩니다.

**📌 [백엔드 API document 보러가기](https://jewoos.site/docs/api)**

### 🛠️ Skills

- `Java17`
- `Spring Boot`, `Spring Security`, `Spring Data JPA`, `Querydsl`
- `MySQL`, `Redis`
- `Nginx`, `Tomcat`
- `Docker`
- `AWS EC2`, `AWS RDS`, `AWS S3`
